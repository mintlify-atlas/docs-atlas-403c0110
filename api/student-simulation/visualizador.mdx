## Overview

The `VisualizadorSimulacion` class handles all graphical aspects of the simulation, including rendering the campus layout, animating student movements, and displaying real-time statistics.

**Key Features:**
- Dual-panel visualization (spatial view + time-series graphs)
- Real-time animation of student movement
- Occupancy tracking with capacity indicators
- Color-coded zones and student markers
- Statistical overlay with current occupancy percentages

#### Implementation

```python
def ejecutar(self):
    anim = animation.FuncAnimation(
        self.fig,
        self._actualizar_animacion,
        frames=self.pasos_simulacion,
        interval=500,  # Milliseconds between frames
        repeat=False,
        blit=False
    )
    
    plt.tight_layout()
    plt.show()
    
    return anim
```

### _dibujar_zonas_campus

<ParamField path="_dibujar_zonas_campus" type="method">
  Internal method that draws the colored zone rectangles and labels on the campus view.
  
  ```python
  def _dibujar_zonas_campus(self)
  ```
  
  **Rendering Steps:**
  1. Iterates through each campus zone
  2. Creates Rectangle patch with zone dimensions
  3. Applies color from `campus.colores_zonas`
  4. Sets 30% transparency (`alpha=0.3`)
  5. Adds centered text label with zone name
  6. Text has white background for readability
</ParamField>

**Source:** `simulacion_movimiento_estudiantes_universidad.py:364-392`

#### Implementation Details

```python
for ubicacion, zona in self.campus.zonas.items():
    # Create rectangle patch
    rectangulo = plt.Rectangle(
        (zona['x'] - zona['ancho']/2, zona['y'] - zona['alto']/2),
        zona['ancho'],
        zona['alto'],
        facecolor=self.campus.colores_zonas[ubicacion],
        edgecolor='black',
        linewidth=2,
        alpha=0.3
    )
    self.ax_campus.add_patch(rectangulo)
    
    # Add label
    self.ax_campus.text(
        zona['x'], zona['y'],
        ubicacion.value,
        ha='center', va='center',
        fontsize=12, fontweight='bold',
        bbox=dict(boxstyle='round', facecolor='white', alpha=0.8)
    )
```

#### Visual Result

### _actualizar_animacion

<ParamField path="_actualizar_animacion" type="method">
  Core animation function called for each frame. Executes simulation step and updates all visualizations.
  
  ```python
  def _actualizar_animacion(self, frame)
  ```
  
  <ParamField path="frame" type="int">
    Frame number provided by matplotlib.animation (0, 1, 2, ...)
  </ParamField>
  
  **Frame Update Process:**
  
  1. **Execute Simulation Logic**
     ```python
     self.campus.actualizar_simulacion()
     self.paso_actual += 1
     ```
  
  2. **Clear Previous Graphics**
     ```python
     self.ax_campus.clear()
     self.ax_estadisticas.clear()
     ```
  
  3. **Redraw Campus View**
     - Draw zone rectangles
     - Plot student positions as scatter points
     - Color by current location (red/blue/green)
     - Configure axes and grid
  
  4. **Update Statistics View**
     - Plot occupancy lines for each location
     - Draw capacity limit lines
     - Add current statistics text box
     - Configure legend and labels
</ParamField>

**Source:** `simulacion_movimiento_estudiantes_universidad.py:394-508`

#### Student Rendering

```python
# Draw each student as colored dot
for estudiante in self.campus.estudiantes:
    # Determine color based on location
    if estudiante.ubicacion_actual == UbicacionCampus.AULA:

    elif estudiante.ubicacion_actual == UbicacionCampus.BIBLIOTECA:

        linewidth=0.5
```

#### Time-Series Graph

```python
if self.paso_actual > 1:  # Need at least 2 points
    pasos = range(1, self.paso_actual + 1)
    
    for ubicacion in UbicacionCampus:
        datos = self.campus.historial_ocupacion[ubicacion][:self.paso_actual]
        
        # Determine line color
        if ubicacion == UbicacionCampus.AULA:

            color = 'blue'
        else:
            color = 'green'

            markersize=3

            linestyle='--',
            alpha=0.5,
            linewidth=1
```

## Complete Usage Example

```python

from matplotlib import animation

# Setup
```

### Interactive Controls

```python
from matplotlib.widgets import Button

campus = CampusUniversitario(num_estudiantes=50)
visualizador = VisualizadorSimulacion(campus, pasos_simulacion=100)

# Add pause/resume functionality
class AnimationController:
    def __init__(self, anim):
        self.anim = anim
        self.paused = False
    
    def toggle_pause(self, event):
        if self.paused:
            self.anim.resume()
        else:
            self.anim.pause()
        self.paused = not self.paused

anim = animation.FuncAnimation(
    visualizador.fig,
    visualizador._actualizar_animacion,
    frames=100,
    interval=500,
    repeat=False
)

controller = AnimationController(anim)

# Add button
ax_button = plt.axes([0.45, 0.01, 0.1, 0.04])
button = Button(ax_button, 'Pause/Resume')
button.on_clicked(controller.toggle_pause)

plt.show()
```

## Design Rationale

### Why Two Panels?

**Spatial View (Left):**
- Shows WHERE students are
- Visualizes movement patterns
- Identifies clustering
- Intuitive understanding of space usage

**Time-Series View (Right):**
- Shows HOW MANY students over time
- Reveals temporal dynamics
- Displays convergence to equilibrium
- Quantitative validation

### Why 500ms Interval?

- 2 frames per second = comfortable viewing speed
- Fast enough to show dynamics
- Slow enough to observe individual steps
- Matches human perception of change

### Why Full Redraw (blit=False)?

- Simpler implementation
- Handles dynamic text/labels
- More reliable across platforms
- Performance adequate for this simulation scale

### Why Transparency (alpha=0.3 zones, 0.6 students)?

- Zones visible but not overwhelming
- Students stand out clearly
- Overlapping students still visible
- Professional aesthetic
