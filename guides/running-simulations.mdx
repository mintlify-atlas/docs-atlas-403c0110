## Overview

This guide covers multiple ways to run the agent-based modeling simulations in this project. You can execute simulations via command line, VS Code, or directly from Python scripts.

## Available Simulations

The project includes two main simulations:

<CardGroup cols={2}>
  <Card title="Student Movement" icon="users">
    Simulates student movement across campus locations (classroom, library, cafeteria)
  </Card>
  <Card title="Traffic Intersection" icon="traffic-light">
    Models traffic flow through a 4-way intersection with traffic lights
  </Card>
</CardGroup>

## Running from Command Line

<Steps>
  <Step title="Navigate to source directory">
    ```bash
    cd workspace/source
    ```
  </Step>

  <Step title="Run the student simulation">
    <CodeGroup>
      ```bash Default Parameters
      python run_sim.py
      ```

      ```bash Custom Parameters
      python run_sim.py --num 80 --pasos 150
      ```

      ```bash Help
      python run_sim.py --help
      ```
    </CodeGroup>

    <ParamField path="--num" type="integer" default="50">
      Number of students to simulate
    </ParamField>

    <ParamField path="--pasos" type="integer" default="100">
      Number of simulation steps to run
    </ParamField>
  </Step>

  <Step title="Run the traffic simulation">
    <CodeGroup>
      ```bash Default Parameters
      cd "PARCIAL 1/simulation/trafico en una ciudad"
      python run_traffic.py
      ```

      ```bash Custom Steps
      python run_traffic.py --pasos 300
      ```

      ```bash Help
      python run_traffic.py --help
      ```
    </CodeGroup>

    <ParamField path="--pasos" type="integer" default="200">
      Number of simulation steps (frames) to execute. The grid size is fixed at 20x20 cells.
    </ParamField>
  </Step>
</Steps>

## Running from VS Code

You can use VS Code's built-in debugging to run simulations interactively.

<Steps>
  <Step title="Open the simulation file">
    Navigate to the simulation script:
    - Student: `PARCIAL 1/simulation/movimiento estudiantes universidad/simulacion_movimiento_estudiantes_universidad.py`
    - Traffic: `PARCIAL 1/simulation/trafico en una ciudad/simulacion_trafico_ciudad.py`
  </Step>

  <Step title="Press F5 to run">
    The simulation will execute with default parameters and display the matplotlib animation window.
  </Step>

  <Step title="Modify parameters in code (optional)">
    Scroll to the bottom of the file and edit the function call:

    ```python Student Simulation
    ejecutar_simulacion(
        num_estudiantes=80,  # Change this
        pasos=150           # Change this
    )
    ```

    ```python Traffic Simulation
    ejecutar_simulacion(
        ancho_ciudad=25,  # Change this
        alto_ciudad=25,   # Change this
        pasos=300        # Change this
    )
    ```
  </Step>
</Steps>

## Running from Python Shell

You can import and run simulations programmatically from an interactive Python session.

<CodeGroup>
  ```python Student Simulation
  import sys
  from pathlib import Path
  
  # Add source directory to path
  sys.path.insert(0, str(Path('workspace/source/PARCIAL 1/simulation/movimiento estudiantes universidad')))
  
  import simulacion_movimiento_estudiantes_universidad as sim_student
  
  # Run with custom parameters
  campus, viz, anim = sim_student.ejecutar_simulacion(
      num_estudiantes=60,
      pasos=120
  )
  ```

  ```python Traffic Simulation
  import sys
  from pathlib import Path
  
  sys.path.insert(0, str(Path('workspace/source/PARCIAL 1/simulation/trafico en una ciudad')))
  
  import simulacion_trafico_ciudad as sim_traffic
  
  # Run with custom parameters
  sim_traffic.ejecutar_simulacion(
      ancho_ciudad=30,
      alto_ciudad=30,
      pasos=250
  )
  ```
</CodeGroup>

</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="ModuleNotFoundError">
    Ensure you're running from the correct directory and all dependencies are installed:

    ```bash
    pip install matplotlib numpy
    ```
  </Accordion>

  <Accordion title="Animation window doesn't appear">
    This can happen in some environments. Try setting the matplotlib backend:

    ```python
    import matplotlib
    matplotlib.use('TkAgg')
    import matplotlib.pyplot as plt
    ```
  </Accordion>

  <Accordion title="Simulation runs too fast/slow">
    Edit the `interval` parameter in the `FuncAnimation` call within the simulation file:

    ```python
    anim = animation.FuncAnimation(
        self.fig,
        self._actualizar_animacion,
        frames=self.pasos_simulacion,
        interval=500,  # Increase for slower, decrease for faster
        repeat=False,
        blit=False
    )
    ```
  </Accordion>
</AccordionGroup>
